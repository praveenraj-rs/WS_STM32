# STM32 Bare-Metal CMSIS Learning Repository

This repository is created to **learn STM32 microcontroller programming from scratch using bare-metal CMSIS**, without relying on HAL or CubeMX.

The purpose of this project is to build a **strong foundation in ARM Cortex-M and STM32 peripheral programming** by directly working with registers, startup code, linker scripts, and low-level firmware concepts.

---

## Objectives

- Learn **bare-metal programming** on STM32
- Understand **CMSIS core and device layers**
- Work directly with **register-level programming**
- Build **reusable firmware components**
- Maintain **well-structured and versioned examples**
- Create a long-term reference for future projects

---

## Notes

- This repository is **for learning and experimentation**
- Code is written for **clarity over optimization**
- Register-level access is intentionally explicit
- Examples are incremental and modular
